# Community Launch Kit

## Default Pilot

- 30 learners
- 12 weeks
- 90-minute weekly session
- Optional 60-minute practice lab
- 3 project partners
- 2 facilitators
- 1 demo day
- 1 public learning report

## Best-Fit Hosts

- Community colleges
- Public libraries
- Workforce boards
- Adult education providers
- Nonprofits
- Unions
- Small-business associations
- Employer learning teams

## 10-Day Launch Plan

Days 1-2: Name the anchor

- Pick host
- Name program lead
- Choose learner population
- Choose three role tracks
- Confirm schedule

Days 3-4: Recruit project partners

- Ask for one safe, recurring workflow from each partner
- Confirm scoping call and demo day availability
- Avoid high-stakes or sensitive-data projects

Days 5-6: Confirm facilitators

- Pair one AI-fluent operator with one local trust lead
- Assign learner support
- Review privacy and accessibility defaults

Days 7-8: Invite learners

- Use plain language
- Emphasize that no technical background is required
- Explain portfolio outcomes
- Ask about access needs

Days 9-10: Set measurement loop

- Create baseline survey
- Create portfolio folders
- Confirm project rubrics
- Schedule midpoint reviews
- Invite demo day panel

## Learner Intake Questions

- What kind of work or learning do you want AI to help with?
- What worries you most about AI?
- What tools have you already tried?
- What would make this cohort accessible for you?
- What role, job, or community contribution would you like this to support?
- Do you have access to a device and internet for practice?

## Baseline Measures

Ask learners to rate from 1 to 5:

- I can explain what AI tools are good and bad at.
- I can write a prompt that gives useful context.
- I can check AI outputs for accuracy.
- I can decide when not to use AI.
- I can improve a work or learning task with AI.
- I can explain my AI use to another person.

Collect:

- Role interest
- Prior AI use
- Access needs
- Portfolio goal

## Cohort Report Outline

Publish after Week 12:

1. Who participated
2. What project partners brought
3. What learners built
4. What improved
5. What failed or needs caution
6. What learners want next
7. What employers or partners want next
8. What to change for the next cohort

## Minimum Materials

- Curriculum
- Facilitator guide
- AI use policy
- Project brief template
- Portfolio rubric
- Partner ask
- Learner invite
- Demo day agenda
